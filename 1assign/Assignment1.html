Math 151A: Assignment 1

Assigned Monday Jan. 8, 2018, due Friday Jan. 12, 2018

The purpose of this assignment is to have you become familiar with the software programs that you will be using to do the computational assignments for this class. You will be using Matlab. Matlab is available in the PIC lab (2000 Math Sciences). For home use, one can purchase a student edition of Matlab online or one can use the Matlab clone GNU octave that is available for Windows, Linux and Mac OS systems.

In this assignment you will download, modify, and then execute a script (an m-file) that plots \(\sin(x^2)\) for \(x \in [0, \, 2 \pi ]\). You will then save the plot as a pdf file. To turn in this assignment, you will upload your modifed script and the pdf of the plot to the CCLE course website

[0] Verify that using Matlab (or Octave) you can create and save a plot of \(\sin(x)\) for \(x \in [0, \, 2 \pi ]\).

  1. Create a class directory for your all your 151A assignments, and then a subdirectory for Assignment 1. Download Assignment1.m to your Assignment 1 sub-directory.

  2. Start up Matlab or Octave and change the directory to your Assignment 1 sub-directory.

  3. Using the command prompt, run the script Assignment1.m:

    1. If you are using the Matlab or Octave IDE, open the file in the editor and select "Run"

      -or-

    2. Type the name of the script (m-file) without the .m extension, Assignment1, at the command prompt, e.g.

      >> Assignment1

  4. Create a *.pdf file of the resulting plot with the name Sin.pdf. .

    1. From the plot window, select File/Save As and specify "Sin.pdf"

      -or-

    2. Execute the command

      >> print('Sin.pdf','-dpdf')
